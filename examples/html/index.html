Theme Switcher

Switch themes by changing the theme CSS import in the HTML head:

theme-switching.html 
<!-- Pink Theme -->
<link rel="stylesheet" href="themes/pink.css">

<!-- Blue Theme -->
<link rel="stylesheet" href="themes/blue.css">

<!-- Purple Theme -->
<link rel="stylesheet" href="themes/purple.css">

<!-- Green Theme -->
<link rel="stylesheet" href="themes/green.css">

<!-- Greyscale Theme -->
<link rel="stylesheet" href="themes/greyscale.css">

Try changing the theme link in this file's head to see different color schemes!
