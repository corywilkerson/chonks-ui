Quick Start

Plain HTML/CSS

index.html 
<link rel="stylesheet" href="chonks.css">
<link rel="stylesheet" href="themes/pink.css">

<button class="chonks-btn">Hello!</button>

No build tools required. Just link and go.

React

App.jsx 
import { Button, useTheme, THEMES }
from 'chonks-ui/react'

useTheme(THEMES.PINK)
<Button>Hello!</Button>

Modern React components with hooks.
